TUTORIAL

2. Buka Chrome / Edge.
3. Masuk ke: chrome://extensions atau edge://extensions
4. Aktifkan Developer mode.
5. Klik Load unpacked dan pilih path folder dimana lu clone project ini
6. login servant kek biasa
7. buka menu ticket list
8. klik extensionnya tunggu dia berhasil get header request
9. klik tarik tiket abis tu langsung copy atau bisa teken button download txt buat download
